# MBI Dark Portfolio

A static HTML/CSS/JS portfolio website for MD Ismail Foraji (MBI Dark) — Web Designer, Film Maker, Storyteller, Videographer.

## Project Structure

- `index.html` — Main landing page with preloader, hero, and sections
- `about.html` — About page
- `projects.html` — Projects showcase page
- `contact.html` — Contact page
- `style.css` — All styles
- `script.js` — All JavaScript (animations, interactions)
- `projects.json` — Project data
- `CV_MD_Ismail_Hossen.pdf` — Downloadable CV
- `server.js` — Simple Node.js static file server for development

## Running the Project

The project is served via a Node.js static HTTP server on port 5000 (0.0.0.0).

**Start command:** `node server.js`

## Deployment

Configured as a static site deployment with `publicDir: "."`.
